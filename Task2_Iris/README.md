# Iris Flower Classification

## Objective
Predict the species of Iris flower using Machine Learning.

## Steps Followed
1. Data Loading
2. Data Cleaning
3. Feature Selection
4. Model Training
5. Model Evaluation

## Model Used
Logistic Regression
## Project Description

This project classifies iris flowers into different species using Logistic Regression.
The model was trained using sepal and petal measurements from the Iris dataset.
