# Sales Prediction using Machine Learning

## Objective
Predict product sales based on advertising spend.

## Steps Followed
1. Data Loading
2. Data Cleaning
3. Feature Selection
4. Model Training
5. Model Evaluation

## Model Used
Linear Regression
## Project Description

This project predicts product sales based on advertising budgets using Linear Regression.
The model analyzes TV, Radio, and Newspaper advertisement spending to estimate sales performance.
